# Mutual Fund Analytics Platform

## Overview

This project was developed as part of the Bluestock Fintech Capstone Internship Program.

The platform analyzes mutual fund industry data, evaluates fund performance, performs risk analytics, and provides interactive dashboard visualizations.

---

## Features

* ETL Pipeline
* Data Cleaning & Transformation
* Exploratory Data Analysis (EDA)
* Performance Analytics
* Risk Metrics (VaR, CVaR, Sharpe Ratio)
* Investor Analytics
* Fund Recommendation Engine
* Power BI Dashboard
